# 🚪 Unity Procedural Backrooms Prototype

<p align="center">
  <img src="./ReadmeAssets/playmode_preview.gif" alt="Backrooms preview" />
</p>

A prototype developed in Unity focused on **procedural room generation** with Binary Space Partitioning (BSP), inspired by the unsettling and endless atmosphere of the Backrooms.

The project explores modular environment creation with procedural room layouts, and immersive horror atmosphere design.

<p align="center">
  <img src="./ReadmeAssets/generation_preview.gif" alt="Generation preview" />
</p>

---

## 🧩 Technologies Used

- `Unity Engine (C#)`
- `Procedural Generation` – Dynamic room spawning and layout assembly (with customization)
- `DOTween` – Used for Camera Headbob Effect
- `FMOD` – Ambience noise and Footsteps
- `Post Processing / Shader Graph` – Visual tweaks to match the original Backrooms aesthetic

---

## 🚀 How to Run

1. Clone the repository  
2. Open the project in Unity  
3. Open the main scene
4. Press Play Mode 🎮  

---

## 👁️ Final Notes

This project is a **non-commercial prototype created for learning and experimentation**, with emphasis on procedural generation, atmosphere design, and immersive horror gameplay systems.
